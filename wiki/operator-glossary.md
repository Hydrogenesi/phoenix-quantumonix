# Operator Glossary

## Core Operators

### ALPHA_OP
Initiation field operator. Opens a new recursion cycle and
establishes the proto-depth formation layer.

### DELTA_OP
Transformation gate. Applies state transitions within the
current depth band.

### SIGMA_OP
Stabilization layer. Monitors the stability metric and applies
corrections to keep the system within the convergence envelope.

### OMEGA_OP
Closure seal. Finalizes a recursion cycle and prepares state
for the next iteration.

### KAPPA_OP
Recursion anchor. Prevents unbounded depth growth by enforcing
the Recursion Depth Law at critical transitions.

### TAU_OP
Temporal binding operator. Synchronizes state across recursion
cycles and maintains memory pattern consistency.

### LNS_OP
Lineage node operator. Records the causal chain of state
transitions for audit and recovery.

### CROWN
Apex operator. Activates only at maximum coherence. Signals
readiness for canon formation.

### BIND
Coherence lock. Freezes the current state signature to prevent
drift during multi-engine operations.

### MERGE
Triadic integrator. Combines three independent state streams
into a unified coherence field.

### FORK
Branch initiator. Opens a parallel recursion path for
exploration without affecting the main depth band.

### REFLECT
Mirror gate. Projects current state back through prior cycles
to detect drift or regression.

### ELEVATE
Ascension trigger. Promotes the system to the next depth band
when all stability conditions are met.

### SEAL
Canon finalizer. Permanently records the current state as
canonical. Cannot be undone.
